Nombre de la Herramienta : Editor de Niveles Pro

Integrantes :
-Axel Chavez Campos
-Josué González Agüero

Esta herramienta será utilziada para la creación y edición de niveles en nuestro proyecto de juego personal. Será capaz de 
agregar e interconectar niveles, a la vez de llenarlos de texturas con normal Maps , agregar soporte de colisiones físicas y 
planear posicionamiento de la cámara.


Puntos cubiertos:
- Diseño y dibujado de la interfaz dentro del proyecto.
- Estructura base de la clase Data con todas las variables y clases necesarias para el almacenamiento ordenado de información.
- Exportación de una estancia de Data en ejecución a un archivo JSON

Por hacer:
- Programar elementos de la interfaz para habilitar la edición de data.
- Importación: crear una nueva estancia de Data a partir de un archivo JSON
